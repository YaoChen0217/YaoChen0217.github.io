服务器介绍

欢迎来到我们的Minecraft生存服务器！这是一个主打友好氛围、长期稳定的纯净生存服，支持多人联机，我们鼓励玩家合作建造、探索世界，共同打造一个温馨的游戏社区。

基本信息

  • 📌 服务器版本：1.20.4
  • 🌍 服务器类型：纯净生存 + 轻度插件
  • 🔌 在线状态：7×24小时稳定运行
  • 👥 最大人数：50人

如何加入

1. 服务器地址

mc.yourserver.com

2. 加入步骤

  1. 打开Minecraft启动器，选择 1.20.4 版本启动游戏
  2. 进入多人游戏，点击 添加服务器
  3. 粘贴服务器地址并保存，点击 加入服务器
  4. 首次进入请阅读欢迎信息并完成验证
